What is an enumerable?

It is a Ruby module providing a set of methods that can be called on different types of objects. These built in methods can be used in lieu of loops when iterating over arrays and hashes. Each of these methods depend on the presence of the #each method which applies a block of code to each element in the array.

Show me some examples

#each

This simply says, for each element in the array, execute this block of code. In this case, it puts 1, 2 and 3. The each method is the basis for the other more complicated enumerables.

#any?

Checks to see if any elements in the provided array pass the test provided in the code block.

#all?

Checks to see if all elements in the provided array pass the test provided in the code block.

#select

Returns the elements in the array that pass the block of code.

#detect

Returns the first element in the array that passes the block of code.

#sort

This method sorts each element in the array from smallest to largest unles otherwise specified.
